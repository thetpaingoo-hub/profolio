# Thet Paing Oo — Portfolio (GitHub Pages)

This is a lightweight, no-build portfolio website (HTML/CSS/JS).

## Quick deploy (GitHub Pages)

1. Create a new repo on GitHub:
   - Option A (personal site): **USERNAME.github.io** (recommended)
   - Option B (project site): any repo name

2. Upload these files to the repo (keep the same structure).

3. In GitHub:
   - **Settings → Pages**
   - **Source**: Deploy from a branch
   - Branch: **main** (or master), Folder: **/** (root)
   - Save

4. Your site will appear at:
   - Personal site: `https://USERNAME.github.io`
   - Project site: `https://USERNAME.github.io/REPO_NAME`

## Customise

- Update your GitHub/LinkedIn links in `index.html`.
- Replace `assets/profile.png` with your preferred photo/avatar.
- Replace `assets/resume.pdf` with your latest CV.

## Local preview

Just open `index.html` in a browser.
